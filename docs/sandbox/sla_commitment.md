# Code Sandbox SLA 承诺书

## 服务级别目标
- 可用性: 99.9%（月度）
- 故障转移时间: <= 30s
- 数据备份 RPO: <= 5min
- 审计日志保留: >= 180 天

## 性能目标
- 冷启动时延: <= 800ms（p95）
- 并发处理能力: >= 1000 实例
- 资源扩展策略: CPU 利用率 <= 80% 时触发横向扩容

## 安全目标
- 默认隔离: 容器只读根文件系统 + 无外联网络
- 权限最小化: drop capabilities + no-new-privileges
- 风险拦截: 越权文件访问/子进程/敏感 API/外联行为实时阻断

## 违约补偿
- 当月可用性低于 99.9%:
  - 99.0%~99.9%: 10% 服务费用减免
  - <99.0%: 25% 服务费用减免
